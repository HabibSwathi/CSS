CSS selectors

we have some css selectors to style the html elements.

  • Universal selectors (*)
  • Element selectors (tag name)
  • Class name selectors (class name using .)
  • Id Selectors (using id attribute #)
  • attribute selectors (using any attribute inside the elements)
  • attribute with value selectors (attribute with values)

Universal Selectors (*)

This selector is for all the elements present in parent Element

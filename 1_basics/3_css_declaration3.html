CSS DECLARATION

We have 3 types of css declaration

  1. Inline css using style attribute inside the html tag.
  2. Internal css using style attribute inside the head tag.
  3. External css using External file by creating using .css extension.

1. External css using External file by creating using .css extension.

The External css is used to apply the styles for multiple webpages or documents.
